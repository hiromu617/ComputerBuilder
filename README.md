# ComputerBuilder
コンピュータを構築し、最終的なスコアに基づいて性能を比較するアプリケーション

# dev

```bash
$ git clone https://github.com/hiromu617/ComputerBuilder.git
$ cd ComputerBuilder
$ npm install
$ npm run build
```
